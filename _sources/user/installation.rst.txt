Installation
============

You can install napari-bigfish via pip:

	pip install napari-bigfish

Alternatively you can install napari-bigfish from napari. Start napari and run the command ``Install/Uninstall Plugins...`` from the menu ``Plugins``. Find napari-bigfish in the list of available plugins and press the ``install``-button.
